# V1 SNN 实验报告 - Obsidian Vault

## 使用说明

### 导入到 Obsidian

1. 解压 `v1_snn_experiment_report.zip`
2. 打开 Obsidian
3. 点击 "Open folder as vault"
4. 选择解压后的 `obsidian_report` 文件夹

### 目录结构

```
obsidian_report/
├── V1 SNN 训练方案对比实验报告.md   # 主报告
├── README.md                        # 本文件
├── attachments/                     # 图表附件
│   ├── fig1_loss_comparison.png
│   ├── fig2_accuracy_comparison.png
│   ├── fig3_overfitting_analysis.png
│   ├── fig4_spike_rate_evolution.png
│   ├── fig5_training_speed_comparison.png
│   ├── fig6_experiment_overview.png
│   ├── fig7_bcoo_learning_curves.png
│   └── fig8_radar_comparison.png
└── scripts/                         # 生成脚本
    └── detailed_experiment_report.py
```

### Obsidian 设置建议

为了最佳显示效果，建议在 Obsidian 设置中：

1. **Files & Links** → 将 "Default location for new attachments" 设为 "In subfolder under current folder"，并设置为 `attachments`

2. **Editor** → 启用 "Strict line breaks" 以正确显示表格

3. 安装推荐插件：
   - Dataview (用于高级查询)
   - Kanban (用于任务管理)

### 功能特性

- 使用 Obsidian Callouts 突出重要信息
- Wiki-style 内部链接
- 可视化图表嵌入
- 任务清单跟踪
